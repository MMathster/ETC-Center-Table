ETC Loci Sandbox — Thales Configuration

\(A=(1,0)\) and \(B=(-1,0)\) are fixed while \(C\) moves on the unit circle. The loci use exact rational parametric forms \(x(t),y(t)\) with \(t=\tan(\theta/4)\), obtained by normalizing barycentric weights \(\bar a = 1-t^2\), \(\bar b = 2t\), \(\bar c = 1+t^2\) and applying the Weierstrass half-angle identities.
