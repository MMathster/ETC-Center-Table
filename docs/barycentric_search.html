Encyclopedia of Triangle Centers

Barycentric Search Engine Live ETC extraction • clickable rows • JSXGraph preview

The table below now treats the upstream ETC pages as the source of truth. It probes ETC.html, then ETCPart2.html, ETCPart3.html, and so on until ETCPart37.html is missing; every genuine barycentric variant found for a center is kept, searchable, and selectable for plotting in the Thales-style preview board.

Notebook-style parsing

The loader walks the same header block structure as the notebook, rejects prose and cross-references, and keeps helper function definitions when a barycentric line uses f(a,b,c)-style shorthand.

Page-aware filtering

Each extracted center stores its ETC source page, so you can search by ID, name, page, or barycentrics and immediately see which part of the encyclopedia it came from.

Clickable plotting

Click any result row to attempt a numerical Thales-circle plot. When a coordinate is evaluable, the preview board shows the selected center, its locus, and the current slider-controlled position.

Selected center preview

Click a table row to load the selected center into this board. The plot uses the same Thales configuration as the introduction page: A=(1,0), B=(-1,0), and C=(\cos\theta,\sin\theta).
